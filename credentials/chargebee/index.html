# Chargebee

You can find information about the operations supported by the Chargebee node on the integrations page. You can also browse the source code of the node on GitHub.

# Prerequisites

Create a Chargebee account.

# Using Access Token

  1. Open your Chargebee dashboard.
  2. Click on "Settings".
  3. Click on "Configure Chargebee".
  4. Scroll down and click on "API Keys".
  5. Use selected API key and Account name in your Chargebee node credentials in n8n.
